CONCOLIC TEST REPORT
Hàm thực hiện:
public static int abs(int x){
  if (x < 0)   return -x;
 else   return x;
}

STEP 1: Clone hàm cần kiểm thử vào một file java có thể chạy được: core-engine\cfg\src\main\java\data\CloneFile.java
STEP 2: Sinh cây CFG dựa trên hàm được truyền vào
STEP 3: Sinh dữ liệu ngẫu nhiên cho các parameter [int]: [8]
STEP 4: Chạy dữ liệu ngẫu nhiên đấy, lưu những câu lệnh đã được chạy qua: [1 x < 0 false, 2 return x;
 ]
STEP 5: Tìm node chưa được phủ: CFGNode{, content='return -x;
', isRootNode=false, isEndNode=false}
STEP 6: Sinh đường thi hành từ node chưa được phủ đấy
STEP 7: Thực thi tượng trưng đường thi hành và sinh test data tương ứng: [-1]
STEP 8: Đánh dấu những câu lệnh (node) đã chạy qua sau khi thực hiện chạy hàm với dữ liệu vừa được sinh: [1 x < 0 true, 1 return -x;
 ]
