## 2. Run

#### 2.1. Prerequisites
- MySql (version 8.x.x)
- JDK 11
- Apache Maven (version 3.x.x)

#### 2.2. PostInstall
Open mysql and create 2 databases: 

- micro_jcia_user: ```create database micro_jcia_user```
- micro_jcia_project: ```create database micro_jcia_project```

#### 2.3. Install

##### 2.3.1. Naive
Download project and using services tool in Intellij to run 

##### 2.3.2. Docker (Install Docker and docker-compose first)
First step:

- if linux, then run: ```.\build.sh```
- else windows, then run: ```.\build.bat```

Second: ```docker-compose up```
