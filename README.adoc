JCIA is Java Change Impact Analysis software help you to easily determine all dependency in your Java Project. JCIA also helps to compare project version, calculate dependency impact, visualize your Project into clear diagrams.

## Service

### All Service Ports
- port 8761: Discovery-Service

#### Administration
- port 8001: Utility-Service
- port 8002: User-Service
- port 8003: Project-Service

#### Parse Service
- port 7001: Parser-Service
- port 7002: Java-Service
- port 7003: Spring-Service
- port 3001: Xml-Service

#### Algo Service
- port 6001: Cia-Service
- port 6002: VerControl-Service

#### Support Service
- port 5001: File-Service

#### Version Control Service
- port 4001: Github-Service

#### Run
- ```.\build.bat``` for the first time
- ```docker compose up```
